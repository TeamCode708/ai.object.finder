AI Object Finder

Welcome to AI Object Finder web-app!

This web-app is made for an animal rescue organization, Protractor. It works with an aim to give home to all the homeless animals in the state. They'll be needing help of video surveillance department to tell them if they find any such animals on they webcam. To make their task easy, we've launched this web-app, using which they can easily find homeless animals. You simply have to type the name of the animal in the search bar and it'll detect that particular animal. This web-app uses an object detecting model called CoCoSSD, which aims to find multiple objects in a single image.
NOTE : Search the animal by the name present in the CoCoSSD Model.
